Шаг 10: AI-рендер кухни

Что исправлено:
1. app.js больше не использует жесткий адрес сайта. API вызывается через текущую папку проекта: api.php.
2. Загрузка картинки шага 10 идет в uploads/ через api.php?action=upload.
3. Генерация идет через api.php?action=render.
4. Размеры рендера ограничены рабочими вариантами: 1024x1024, 1536x1024, 1024x1536.
5. Добавлены понятные ошибки: нет API-ключа, нет PHP cURL, файл не найден, проект открыт как file://.

Куда вставить ключ:
Откройте config.php и вставьте ключ в строку:

  'OPENAI_API_KEY' => 'ВАШ_КЛЮЧ_ТУТ',

Строку модели оставьте так:

  'OPENAI_IMAGE_MODEL' => 'gpt-image-1',

Важно:
- Шаг 10 не будет работать, если просто открыть index.html двойным кликом с компьютера.
- Нужно залить проект на сайт с PHP или открыть через локальный PHP-сервер.
- Папки projects/ и uploads/ должны быть доступны для записи.
- На сервере должен быть включен PHP cURL.
